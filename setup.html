The login page downloads because you need a PHP server to run this website. Follow the steps below to set it up properly.

Step 1: Install XAMPP (Recommended)

Download and install XAMPP from: https://www.apachefriends.org/

XAMPP includes Apache web server, PHP, and MySQL - everything you need!

Step 2: Start Services

1. Open XAMPP Control Panel

2. Start Apache and MySQL services

3. Make sure both show "Running" status

Step 3: Copy Files

1. Copy all website files to: C:\xampp\htdocs\eduhub\

2. Make sure the folder structure looks like this:
